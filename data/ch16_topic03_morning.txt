TOPIC: C++17 Template Improvements

Advanced C++17 features that simplify template code

What this carousel covers:
→ Class Template Argument Deduction (CTAD) - eliminates redundant type specifications (std::pair(42, 3.14) vs std::pair<int, double>(42, 3.14))
→ Custom deduction guides for controlling type deduction in user-defined templates
→ 4 types of fold expressions - unary/binary left/right folds for parameter packs
→ Fold expression with 30+ supported operators (arithmetic, logical, bitwise, comparison)
→ Empty parameter pack behavior (&&→true, ||→false, comma→void(), others→ill-formed)
→ constexpr lambdas - implicitly constexpr in C++17 for compile-time computation
→ Parallel algorithms with std::execution policies (seq, par, par_unseq)
→ std::scoped_lock for deadlock-free multi-mutex locking
→ std::to_chars/from_chars - 3-10x faster than stringstream, no allocation
→ Map improvements: insert_or_assign and try_emplace (never moves key)
→ Nested namespace syntax (namespace company::autonomous::perception)
→ [[nodiscard]], [[maybe_unused]], [[fallthrough]] attributes

Key insights:
◆ CTAD works through 5-step mechanism: constructor examination → argument matching → type deduction → deduction guide lookup → instantiation
◆ CTAD fails with non-deducible parameters, ambiguous constructors, private constructors, or copy-list initialization
◆ Fold expressions eliminate recursive template instantiation for variadic templates
◆ Example speedup: processing 1,000,000 point cloud with std::execution::par achieved 3.75x speedup (45ms → 12ms)
◆ Parallel algorithms require thread-safe operations - use std::atomic for shared state
◆ std::scoped_lock prevents deadlock even when threads lock same mutexes in different order
◆ std::to_chars benefits: 3-10x faster, no dynamic allocation, locale-independent, no exceptions, round-trip guarantee
◆ constexpr lambdas enable zero-overhead compile-time lookup tables (e.g., steering angle tables)
◆ try_emplace only constructs value if insertion happens (more efficient than operator[])
