Key insights:
в—† SIGPIPE signal crashes your process when writing to closed socket - use MSG_NOSIGNAL flag or signal(SIGPIPE, SIG_IGN)
в—† send() and recv() don't guarantee all bytes transferred - implement send_all() and recv_all() wrapper functions with loops
в—† Port enters TIME_WAIT state (60 seconds) after crash - use setsockopt(SO_REUSEADDR) to allow immediate restart
в—† Forgetting htons() on little-endian: port 5000 (0x1388) becomes 34835 (0x8813) - always convert with htons/htonl
в—† File descriptor limit typically 1024 (ulimit -n) - socket leaks cause EMFILE errors, use RAII SocketGuard class
в—† TCP keep-alive options: TCP_KEEPIDLE (idle time), TCP_KEEPINTVL (interval), TCP_KEEPCNT (probe count)
в—† UDP header 8 bytes vs TCP 20+ bytes - use UDP for sensor streams (lidar), TCP for critical data (HD maps)
в—† Byte order conversion: htons() for 16-bit ports, htonl() for 32-bit IPs, ntohs/ntohl to convert back
